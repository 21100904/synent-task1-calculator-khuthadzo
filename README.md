# synent-task1-calculator-khuthadzo
Simple Calculator CLI in Python



## 📌 Description
This is a command-line calculator built using Python. It performs basic arithmetic operations.

## ⚙️ Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles invalid input
- Prevents division by zero

## ▶️ How to Run
1. Install Python
2. Open terminal in project folder
3. Run:
   python calculator.py
